- **Diseño de lujo internacional** con paleta navy + gold
- **Animaciones fluidas**: aviones, nubes, partículas, AOS, counters
- **Carrito de servicios inteligente** → finaliza en WhatsApp con resumen

- **ElhyBot** (chatbot IA simple) con respuestas sobre visas, seguros, etc.
- **CTA constantes** y optimizado para conversión
- **SEO completo**: Meta tags, Open Graph, Schema.org TravelAgency, Sitemap, Robots
- **100% Responsive** + Accesible
